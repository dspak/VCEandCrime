Virtual Civic Engagement and Crime

Correlating SeeClickFix use with crime in New Haven, CT

Download this project as a .zip file Download this project as a tar.gz file

This is the code related to the analysis of SeeClickFix usage and Crime in New Haven, CT. Contained are the methods for assigning SeeClickFix and Crime data to neighborhoods and aggregated by time intervals. Included are summary figures for each of the data types and an interrupted time series model. Raw data that are processed by these scripts will be included soon.
